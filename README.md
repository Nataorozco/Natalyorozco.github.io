# Proyecto Web Básica

Proyecto académico personal desarrollado en HTML, CSS y JavaScript como parte de mi formación en Análisis y Desarrollo de Software Talento Tech – Colombia.

## Páginas del proyecto
- `index.html`: Página principal
- `Nosotros.html`: Información sobre el proyecto o el equipo
- `Contactos.html`: Formulario de contacto
- `Calculadora.html`: Calculadora básica en JavaScript
- `Grafico.html`: Página con gráficos interactivos
- `Leer-mas.html` y `Que-es.html`: Información adicional

## Tecnologías utilizadas
- HTML
- CSS
- JavaScript

## Aprendizajes
- Estructuración de páginas web
- Manejo de formularios y botones
- Uso básico de JavaScript para lógica y cálculos
- Organización de archivos en un proyecto

## Nota
Este proyecto está pensado como práctica de Frontend. Se puede abrir directamente en un navegador.
